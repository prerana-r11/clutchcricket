## ClutchCricket

ClutchCricket is a cricket analytics engine designed for T20 internationals using the ball-by ball data from Cricsheet.

It has the following features currently-
* Pressure modelling during run chases
* Detects turning points 
* Estimates win probability
* Batting analytics during clutch situations.
* Match visualization along with momentum analysis.

## Project Overview

Traditional cricket statistics like averages and strike rates often fail to capture the actual difficulty of match situations.

ClutchCricket attempts to quantify:
* How much pressure a batting side is under
* Which moments changed the outcome of a match
* Which batters perform well under pressure
* winning chances evolve ball-by-ball
